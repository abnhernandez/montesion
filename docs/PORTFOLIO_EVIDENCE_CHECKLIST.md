# Checklist de Evidencias para Portafolio

## Capturas obligatorias
- Home con identidad visual completa.
- Login/registro con validaciones visibles.
- Formulario de oracion con estados de exito/error.
- Localizador de templos con mapa.
- Vista movil responsive real.

## Capturas tecnicas recomendadas
- Resultado de build exitoso.
- Resultado de lint sin errores bloqueantes.
- Dashboard de deployment (Vercel).
- Fragmento de PR con descripcion y checklist.

## Video demo (2-3 minutos)
1. Problema que resuelve el proyecto.
2. Flujo principal del usuario.
3. Decisiones tecnicas clave.
4. Seguridad y manejo de errores.
5. Resultado e impacto.

## Diagramas sugeridos
- Arquitectura de alto nivel (cliente, Next, Supabase, servicios externos).
- Flujo de autenticacion.
- Flujo de peticion de oracion.

## Enlaces para compartir
- URL de produccion.
- URL del repositorio.
- URL de documentacion tecnica.
- URL de video demo.

## Evidencia de impacto
- Core Web Vitals antes/despues.
- Tiempo de build.
- Paginas principales y su First Load JS.
- Bugs corregidos con referencia de commit.
